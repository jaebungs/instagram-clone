# Instagram browser clone
### Render images and videos. Login with OAuth.
- TailwindCSS
- Video.js
- Responsive images and vidoes rendering. 
- Login with passpost.

### How to use.
##### Starting server - cd server, npm run start
##### Starting client - cd client, npm run start
